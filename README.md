Zoo Virtuel ASSAD

Projet PHP en programmation orientée objet (POO).

Description

Application web pour gérer un zoo virtuel :

Utilisateurs (admin, guide, visiteur)

Animaux

Habitats

Fonctionnalités

Inscription et connexion

Gestion des rôles

Guide doit être approuvé par l’admin

Visiteur actif par défaut

Affichage des animaux avec leur habitat

Filtrage des animaux par habitat

Utilisation de PDO et POO

Technologies

PHP (POO)

MySQL

PDO

HTML / CSS (Tailwind)

Structure

classes/ (Database, Utilisateur, Animal, Habitat)

admin/

pages publiques (inscription, connexion, animaux)
